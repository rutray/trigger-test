Übung: MSSQL INSTEAD OF Trigger

Vervollständigen Sie den Code für einen "Soft-Delete" Trigger. Nutzen Sie die Begriffe aus der Liste unten.

Verfügbare Wörter:
CREATE TRIGGER | INSTEAD OF | UPDATE | SET | WHERE | SELECT | FROM | END
trg_AnstattLoeschen
ON Produkte
DELETE
AS
BEGIN
     Produkte
     IstGeloescht = 1
     ProduktID IN ( ProduktID deleted);

    PRINT 'Deaktiviert statt gelöscht.';
;
